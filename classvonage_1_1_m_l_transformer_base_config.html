Detailed Description

Base class for ML transformers configuration.Background blur transformer configuration class. See example about how to instanciate a new config object instance below.

std::vector<std::string> ml_model_file_paths;
ml_model_file_paths.push_back(selfie_segmentation_model_file_path);
auto video_blur_config = std::make_unique<MLTransformerVideoBlurConfig>(ml_model_file_paths, BlurRadius::kHigh);
