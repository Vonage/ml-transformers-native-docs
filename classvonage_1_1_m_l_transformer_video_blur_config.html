Detailed Description

Audio noise suppression transformer configuration class. See example about how to instanciate a new config object instance below.

std::vector<std::string> ml_model_file_paths;
ml_model_file_paths.push_back(audio_noise_suppression_model_file_I_path);
ml_model_file_paths.push_back(audio_noise_suppression_model_file_II_path);
auto audio_noise_suppression_config = std::make_unique<MLTransformerAudioNoiseSuppressionConfig>(ml_model_file_paths);
